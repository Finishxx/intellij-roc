# Intellij-roc

At the moment in the early stages of development.

- syntax highlighting
- connects to the LSP.

# Contributing

If you are missing some feature or see a bug, feel free to create an issue!

Code contributions are always welcome - just create a merge request. There is a
lot of low-hanging fruit,
see [Custom Language Support](https://plugins.jetbrains.com/docs/intellij/custom-language-support.html?from=jetbrains.org)
and [Custom Language Support Tutorial](https://plugins.jetbrains.com/docs/intellij/custom-language-support-tutorial.html?from=jetbrains.org)
